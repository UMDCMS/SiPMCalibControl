Terminal control
TERMINAL IS LOCKED

Note: not all terminal inputs are supported and likely never will until python.cmd will allow for modification with the GNU readline interface IO stream.

Supported inputs:

  • All printable characters (as you would expected)
  • backspace
  • Left/Right arrow navigation
  • Clt+u: clear current line

Notable missing components (non-exhaustive):

  • tab-completion
  • up/down arrow for history navigation
  • Input for interactive commands
  • clt+c for command interrupt (there is a button for that)
